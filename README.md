# English Learning App

## Giới thiệu
Ứng dụng hỗ trợ luyện nói tiếng Anh thông qua hội thoại và feedback tự động

## Teach Stack
- Backend: Spring Boot
- Frontend: React (Vite)
- Database: MySQL

## Project Structure
```
english-learning-app/ 
├── backend/ # Spring Boot API 
├── frontend/ # React App 
├── docs/ # Documentation 
├── database/ # Database scripts
```

## Hướng dẫn chạy project

### 1. Clone project

```
git clone https://github.com/huutienSE/LapTrinhJava.git 
cd LapTrinhJava
```

### 2. Chạy Backend
Mở IntelliJ và run BackendApplication.java

Backend sẽ chạy tại:
http://localhost:8080

### 3. Chạy Frontend

```
cd frontend 
npm install 
npm run dev
```

Frontend chạy tại:
http://localhost:5173
